UbisensePositioning
http://UbisensePositioning.codeplex.com


Project Description
--------------------

Reading and Writing of Ubisense objects' position and Listening for Ubisense Tag button presses

based on Ubisense's ObjectPosition and DetectButtonPresses samples

Note that if you want to access Ubisense from a remote machine, you need to install and run
Ubisense Client Connector utility at your client machine as described in the Ubisense documentation.
